# SecretShare

> The Firefox marketplace is no longer active. This can still be used as a standard web application

## About
Firefox OS App for sharing secrets with your friends.

## Uses...
* HTML/CSS/JavaScript
* OneTimeSecret API
* IndexedDB to store the API credentials
